# **App Name**: SalaryCraft

## Core Features:

- Pay Level Input: Dropdown for selecting the pay level (1 to 18).
- Basic Pay Input: Input field for the current basic pay amount.
- Month and Year Selection: Selection of the month and year for accurate day calculation.
- Days Worked Input: Input field for the number of days worked in the selected month.
- DA Percentage Input: Input field for the Dearness Allowance (DA) percentage.
- HRA Option: Radio buttons for selecting the House Rent Allowance (HRA) option (10%, 20%, 30%).
- Salary Calculation and Display: Calculates and displays the new basic pay, DA, TA, HPCA, HRA, deductions, gross salary, and net salary based on the provided inputs.

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) to convey stability and professionalism.
- Background color: Light grey (#F0F4F7), almost white.
- Accent color: Green (#8FBC8F) to indicate positive values (like net salary) and CTAs.
- Body and headline font: 'Inter', a sans-serif font for a clean, modern, and readable interface.
- Responsive layout to ensure usability across devices, with a clean and clearly structured output section.
- Use of simple, professional icons to enhance the user experience and visual appeal. Example: a calendar icon next to date fields
- Subtle animations on the 'Calculate Salary' button and output fields to indicate calculation in progress.
